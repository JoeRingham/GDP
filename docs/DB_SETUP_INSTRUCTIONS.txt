get database connection info from config.json.database.main !!

> sudo -su postgres

(if the database that you are trying to create already exists, do this:
	> DROP DATABASE <database>;
)

> CREATE DATABASE <database>;
> CREATE USER <username> WITH PASSWORD '<password>';
> GRANT ALL PRIVILEGES ON DATABASE <database> TO <username>;
> ALTER USER <username> SET search_path='<schema>';

> \q

> node database/buildFiles/buildDB.js main
